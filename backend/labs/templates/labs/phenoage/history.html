{% blocktrans %}We use PhenoAge (Levine et al., 2018), a scientifically validated algorithm that estimates your biological age by combining your chronological age with 9 biomarkers from your blood test. Each marker reflects the state of a different system in your body:{% endblocktrans %}

  • ·{% trans "Albumin" %} — {% trans "hepatic protein." %} {% trans "Decreases with aging." %}
  • ·{% trans "Creatinine" %} — {% trans "kidney function." %} {% trans "Tends to rise with age." %}
  • ·{% trans "Glucose" %} — {% trans "energy metabolism." %} {% trans "Rises with aging and sedentary lifestyle." %}
  • ·{% trans "CRP" %} — {% trans "systemic inflammation." %} {% trans "High CRP ages the result further." %}
  • ·{% trans "MCV" %} — {% trans "red blood cell size." %} {% trans "Changes with age and diet." %}
  • ·{% trans "RDW" %} — {% trans "erythrocyte variability." %} {% trans "Highest weight in calculation; elevated indicates cellular stress." %}
  • ·{% trans "Alkaline Phosphatase" %} — {% trans "bone and liver status." %} {% trans "Rises with tissue damage and age." %}
  • ·{% trans "WBC" %} — {% trans "immune system." %} {% trans "Altered by aging." %}
  • ·{% trans "% Lymphocytes" %} — {% trans "adaptive immune response." %} {% trans "Decreases with age (immunosenescence)." %}

{% blocktrans %}The algorithm combines these 9 values using a formula derived from more than 11,000 people in the NHANES population study. The result is an estimated biological age: if it is lower than your real age, your body is functioning better than expected; if it is higher, there is room for improvement.{% endblocktrans %}

{% blocktrans %}⚠️ This estimate is indicative and educational, not a medical diagnosis. A single value may be affected by recent infections, medication or the time of blood draw. Its greatest utility is tracking over time.{% endblocktrans %}
